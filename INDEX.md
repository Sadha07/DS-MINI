## 💡 Core Concepts (Quick Reference)

### Correlation
Two things moving together. Scale: -1 to +1
- Close to 1 = Strong positive
- Close to 0 = No relationship  
- Close to -1 = Strong negative

### P-value
Probability result is from random chance
- P < 0.05 = Real relationship (trustworthy) ✓
- P ≥ 0.05 = Could be random (not trustworthy) ✗

### R² Score
How much variation the model explains
- 0.8+ = Excellent (explains 80%+)
- 0.5-0.8 = Good
- <0.5 = Poor

### Linear Regression
Creates equation: Prediction = Base + (Coef × Feature1) + (Coef × Feature2)

## 🔍 Key Findings from Analysis

From analyzing 10,000 basketball teams:

**What Drives Wins?**
1. ✅ **Defensive efficiency** (lower rating = more wins)
2. ✅ **Offensive efficiency** (higher rating = more wins)  
3. ✅ **Ball handling** (lower turnovers = more wins)
4. ❌ **3-point shooting** (surprisingly NOT significant)

**Model Performance:**
- Average prediction error: ~15.5%
- Best model: Multiple regression with 12 features
- Can rank teams and identify over/underperformers
